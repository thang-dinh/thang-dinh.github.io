The four subshells: s, p, d, f

Electrons in an atom occupy orbitals — 3D standing-wave solutions to the Schrödinger equation. Each orbital is labeled by quantum numbers (n, ℓ, mℓ). The angular momentum quantum number ℓ determines the subshell and orbital shape:

s (ℓ = 0)

Spherical symmetry. 1 orbital (mℓ = 0). Present in every shell.

Capacity: 2e−

p (ℓ = 1)

Dumbbell-shaped lobes. 3 orientations along x, y, z (mℓ = −1, 0, +1).

Capacity: 6e−

d (ℓ = 2)

Cloverleaf shapes. 5 orientations. First appears at n = 3.

Capacity: 10e−

f (ℓ = 3)

Complex multi-lobed. 7 orientations. First appears at n = 4.

Capacity: 14e−

Each orbital holds at most 2 electrons with opposite spins (Pauli exclusion). So subshell ℓ has (2ℓ+1) orbitals × 2 spins = 2(2ℓ+1) electrons. Shell n contains ℓ = 0, 1, …, n−1:

Total capacity = 2(1) + 2(3) + 2(5) + … + 2(2n−1) = 2n²

The gnomon decomposition

The sum of the first n odd numbers equals n² — a fact the Pythagoreans proved geometrically. Each odd number 2k−1 is a gnomon: the L-shaped layer you add to grow a square by one step.

Two side-by-side squares: spin-up (↑) + spin-down (↓). Every cell = one unique quantum state (n, ℓ, mℓ, ms).

Filling order: 9 o'clock → 3 o'clock sweep

Within each subshell, Hund's rule says electrons first occupy all orbitals singly with parallel spin, then pair up. In the gnomon, this means the left square (+ labels) fills completely before the right (– labels). Each gnomon layer under the left square is labeled with its subshell and “+” for spin-up; the right square uses “–” for spin-down.

Within each square, the L-shaped gnomon fills in a sweeping arc: up the vertical arm (9→12 o'clock on the left), then across the horizontal arm and down (12→3 o'clock on the right). This traces a continuous arch across both squares:

Subshells fill in order of increasing energy (Aufbau principle): 1s, 2s, 2p, 3s, 3p, 4s, 3d, 4p, … roughly following the Madelung (n+ℓ) rule. The spatial ordering of mℓ values within each gnomon is a convention — the orbitals are degenerate (same energy).

What the gnomon does not show

Orbital shapes are not nested gnomons. A p-orbital is not physically wrapped around an s-orbital the way gnomons nest geometrically. This is a combinatorial visualization of quantum state counting, cleanly separated from spatial wavefunctions. The separation is intentional: conflating counting with spatial shape is what makes the Bohr model misleading for multi-electron atoms.
